# Pagination (PHP + MySQL)

## Features
- Displays limited records per page
- Uses LIMIT and OFFSET
- Shows page numbers
- Previous and Next buttons
- Works with any table (posts/users/etc.)

## How to Run
1. Update database credentials in db.php.
2. Run the app:
   http://localhost/pagination/pagination.php
3. Add more posts in the database to test pagination.
